Arborescence des ecrans

app/(app)/meals/
meals/index.tsx — Planning hebdo (ecran principal)
Vue semaine flexible, repas par jour, FAB pour ajouter
meals/recipes/index.tsx — Catalogue de recettes
Recherche, filtres (tags, difficulte, temps), grille de cards
meals/recipes/[id].tsx — Fiche recette detaillee
Photo, ingredients, etapes, portions ajustables, bouton "Planifier"
meals/recipes/create.tsx — Creer une recette perso
Formulaire : titre, ingredients (autocomplete), etapes, temps, tags
meals/add.tsx — Ajouter un repas au planning
Bottom sheet : choisir recette, date, type de repas, cuisinier, portions
meals/history.tsx — Historique des repas
Repas passes, frequence, dernier prepare le...
meals/favorites.tsx — Recettes favorites
Filtrable, acces rapide pour planifier

Flux principaux

Parcours utilisateur
Flux 1 : Planifier un repas
Planning → "+" → Choisir recette (catalogue) → Date + type + cuisinier → Confirmer
→ Cree un meal_plan_item + une tache household "Cuisiner [recette]"
→ Ingredients ajoutes automatiquement a la liste de courses active
Flux 2 : Generer la liste de courses
Planning → Bouton "Liste de courses" → Choisir periode → Generer
→ Cree/met a jour une SharedList "shopping" avec ingredients agreges
→ Fusionne les quantites (300g tomate + 200g tomate = 500g tomate)
Flux 3 : Ajouter une recette perso
Catalogue → "+" → Formulaire recette → Ingredients (autocomplete) → Sauvegarder
→ Recette liee au household, visible par tous les membres
Flux 4 : Suggestions depuis l'historique
Planning → Section "Suggestions" → Recettes frequentes / pas cuisinees depuis longtemps
→ Basees sur meal_plan_items passes du foyer
