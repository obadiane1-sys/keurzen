Points de design

  • Navigation semaine : fleches gauche/droite, swipe horizontal possible
  • Jours flexibles : seuls les jours avec repas ou le jour courant sont affiches en detail
  • Avatar cuisinier : pastille coloree avec initiale (reprend le member color existant)
  • Jours vides : placeholder dashed "Ajouter un repas" cliquable
  • Suggestions : carousel horizontal base sur l'historique, visible surtout quand le planning est vide
  • Actions header : acces rapide au catalogue et a la generation de liste de courses
