# Design Tokens

## Colors
- mint: #88D4A9
- blue: #AFCBFF
- coral: #FFA69E
- lavender: #BCA7FF
- navy: #212E44
- text: #1E293B
- gray-light: #E2E8F0
- bg: #F7F9FC
- white: #FFFFFF

## Radius
- xs: 8
- sm: 12
- md: 16
- lg: 20
- xl: 24

## Shadows
- soft-sm
- soft-md
- soft-lg
All should remain subtle.

## Spacing
Use an 8pt system where possible.

## Typography
- titles: strong, premium, readable
- body: calm and clear
- captions: subtle but legible

## Buttons
- primary: coral
- secondary: mint
- ghost: white / bg with navy text
- destructive: soft red if needed

## Mascot
- house shape
- no arms
- no legs
- soft open eyes
- gentle highlights
